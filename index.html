Futuros Desenvolvedores - Grupo

Destinado a compartilhar e também tornar acessíveis os códigos de modo a facilitar a ajuda mútua diante de dúvidas/problemas e em evoluções dos projetos do grupo FDG, que tem por objetivo ajudar no crescimento e no aprendizado de desenvolvimento de software de seus membros.

View project on GitHub

Bem vindo(a) à Página do FDG (Futuros Desenvolvedores - Grupo).

Sinta-se à vontade para compartilhar conteúdo e também para acompanhar o que os demais integrantes estão fazendo no nosso repositório. Sempre que gerar um novo código ou alterar algum já existente, não esqueça de atualizar o FDG-repository para que seja mantida sempre a versão mais atual e possa ser acessada de qualquer lugar que tenha acesso à web. Se você estiver usando a versão do GitHub para Desktop, simplesmente sincronize (sync) seu repositório e então você verá o novo conteúdo.

Conteúdo para estudo

Primeiramente focaremos no aprendizado para desenvolvimento web, no que utilizaremos o site da Codecademy, que possui conteúdo free muito bom, com uma didática excelente e ferramentas próprias embarcadas no site. É necessário que se registrem antes de iniciar (clique aqui).

A proposta de sequência para aprendizado é a que segue abaixo, baseado nos fundamentos que cada conteúdo oferece e a dependência que uma pode ter da outra:

1) HTML/CSS (Make a Website);

2) Javascript, Jquery, SQL e AngularJS.

O curso de NodeJS vai ter que ser fora do Codecademy, pois este site não oferece essa opção. Encontrei uma opção no youtube, mas ainda a é preciso avaliar: NodeJS - tp developer (videoaulas no youtube)

Complemento

Como podem ver, essa página é criada no site do GitHub. Mas o que é GitHub? é uma plataforma de hospedagem de código para controle de versão e colaboração em equipe para desenvolvimento de software que permite pessoas trabalharem juntas de qualquer lugar, destacando que hoje em dia o trabalho remoto já é uma realidade grande e utilizada principalmente nos grades países. Para utilizar seus recursos é preciso criar uma conta de usuário. Clicando aqui você será redirecionado para a página de criação da conta para acesso ao GitHub. Ao concluir, será preciso nos informar o seu usuário para que possamos adicioná-lo no 'team' e assim consigam hospedar os arquivos que criarem, o que permitirá a ajuda de alguém do grupo na depuração de algum erro e permitindo também que os demais aprendam com o seu trabalho. A Codecademy oferece um curso gratuito para o uso do GitHub: Learn Git.

Se preferir um exemplo mais simples, existe uma página aqui que dá orientação simples da criação de um repositório até fazer um Merge, que nada mais é do que tornar a sua edição na versão oficial/final, o que chamamos de subir a versão mais atual do arquivo(s)/projeto(s).

A microsoft possui cursos online no site dela Microsoft Virtual Academy que é interessante pra pegar as termologias da área de desenvolvimento e também informações sobre os produtos deles: Aprendendo a Programar

Dicas e sugestões

Com tantas opções de linguagens e recursos exitentes atualmente fica difícil tomar uma direção, ainda mais para quem não tem noção de como o mercado dá a batida. A indicação na ordem dada é pensando naqueles que ainda não tem um mercado definido. Linguagens mais estruturadas e mais complexas são normalmente utilizadas por grandes empresas, o que torna difícil a prática de desenvolvimento e em muitos casos exigem ferramentas pagas. As que aqui foram indicadas são gratuitas e um excelente ponto de partida, não por serem simples apenas, muito pelo contrário, pois oferecerem recursos poderosos que permitem desenvolver softwares profissionais e com ferramentas gratuitas, com um conjunto de vantagens como uma fatia grande de mercado, o que facilita muito a aplicação de mercado, com facilidade de encontrar vagas de emprego, dentre muitas outras vantagens que poderiam ser citadas.

Dentro do GitHub é possível encontrar inúmeras fontes de consulta, assim como também para orientação, mas pra muitas delas, é necessário um conhecimento previo, ou seja, tem que ter visto alguma coisa:

  • Bootstrap
  • CSS preprocessadores como o SASS (tem curso no Codecademy)
  • Jquery
  • Angular.js
  • Começando do Zero - desenvolvedor web tecnologia .NET (@rodrigokono)
  • css-social-buttons
Estes foram alguns dos motivos que nos levaram a escolher o GitHub como base para nossa jornada de desenvolvimento, além do que hoje é uma ferramente utilizada largamente no mercado, inclusive para apresentação de trabalhos desenvolvidos, uma prática que tem sido adotada em contratações profissionais na área tecnológica no segmento de desenvolvimento.

Tecnologia é uma área que exige uma busca incessante por novos conhecimentos, pois muitos e variados são os recursos e consequentemente seus conteúdos que evoluem com uma velocidade impressionante, mas para os apaixonados isso não é um peso, pois quanto mais se sabe, mais se quer precisa saber, considerando que o conhecimento hoje é muito vasto em qualquer área.
